# Wavelength (React + TypeScript)

A pass-and-play digital version of the board game Wavelength.

## Features
- Two-player local gameplay
- Alternating psychic / guesser roles
- Adjustable win threshold
- Pure functional state transitions
- Type-safe scoring logic

## Tech Stack
- React
- TypeScript
- Vite

## Running Locally
npm install
npm run dev
